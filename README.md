# fravegaChallenge
Challenge de fravega

# Servicios
* /add
* /find?id=1
* /update?id=1
* /delete?id=1

# Dependencias
* Spring 2.5.4
* Swagger 2 3.0.0
* Actuator
* H2

# Ejecucion de proyecto
```
mvn clean install
java -jar fravegaChallenge.jar
```
